# 智慧城市边缘视频分析平台 - 数据库设计文档

## 一、数据库概览

- **数据库名称**: `edge_video_analysis`
- **字符集**: utf8mb4
- **引擎**: InnoDB

## 二、表结构

### 2.1 t_lamp (灯杆设备表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键ID |
| lamp_code | varchar(50) | 灯杆编号 |
| lamp_name | varchar(100) | 灯杆名称 |
| location | varchar(200) | 位置描述 |
| camera_url | varchar(500) | 视频流地址 |
| led_status | tinyint | LED状态 0-关 1-开 |
| online_status | tinyint | 在线状态 0-离线 1-在线 |
| create_time | datetime | 创建时间 |
| update_time | datetime | 更新时间 |

### 2.2 t_sensor_data (传感器数据表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键ID |
| lamp_id | bigint | 灯杆ID |
| temperature | double | 温度 |
| humidity | double | 湿度 |
| illumination | double | 光照强度 |
| voltage | double | 电压 |
| current | double | 电流 |
| capture_time | datetime | 采集时间 |
| create_time | datetime | 创建时间 |

### 2.3 t_alarm_rule (报警规则表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键ID |
| lamp_id | bigint | 灯杆ID |
| sensor_type | varchar(20) | 传感器类型 |
| upper_limit | double | 上限值 |
| lower_limit | double | 下限值 |
| enabled | tinyint | 是否启用 |
| create_time | datetime | 创建时间 |
| update_time | datetime | 更新时间 |

### 2.4 t_alarm_record (报警记录表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键ID |
| lamp_id | bigint | 灯杆ID |
| sensor_type | varchar(20) | 传感器类型 |
| sensor_value | double | 触发报警的值 |
| alarm_level | tinyint | 报警级别 |
| alarm_message | varchar(500) | 报警描述 |
| alarm_time | datetime | 报警时间 |
| handled | tinyint | 是否处理 |
| create_time | datetime | 创建时间 |

### 2.5 t_inference_record (推理记录表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键ID |
| lamp_id | bigint | 灯杆ID |
| original_image | longtext | 原始图片 |
| processed_image | longtext | 处理后图片 |
| person_count | int | 人数 |
| inference_results | text | 推理结果 |
| inference_time | datetime | 推理时间 |
| create_time | datetime | 创建时间 |

### 2.6 t_control_command (控制指令表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | bigint | 主键ID |
| lamp_id | bigint | 灯杆ID |
| command_type | varchar(20) | 指令类型 |
| command_status | tinyint | 执行状态 |
| create_time | datetime | 创建时间 |
| execute_time | datetime | 执行时间 |

## 三、索引

- t_sensor_data: (lamp_id, capture_time)
- t_alarm_record: (lamp_id, alarm_time)
- t_inference_record: (lamp_id, inference_time)

## 四、Redis缓存

- sensor:latest:{lampId} - 最新传感器数据
- device:heartbeat:{lampId} - 设备心跳 (TTL 30s)
- device:online:{lampId} - 设备在线状态 (TTL 60s)
