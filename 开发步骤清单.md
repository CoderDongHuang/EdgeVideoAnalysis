# 智慧城市边缘视频分析平台 - 详细开发步骤

## 阶段一：基础框架搭建

### 步骤1.1：创建Spring Boot项目
- [ ] 使用Spring Initializr创建项目
  - Group: `com.edgevideoanalysis`
  - Artifact: `edge-video-analysis`
  - Java版本: 11或17
  - 依赖: Spring Web, Spring Data Redis, MyBatis Framework, MySQL Driver, Lombok, WebSocket

### 步骤1.2：配置pom.xml
- [ ] 添加MyBatis-Plus依赖（版本3.5+）
- [ ] 添加FastJSON2依赖（JSON处理）
- [ ] 添加Hutool依赖（工具类）
- [ ] 添加FFmpeg依赖（java-cv或自定义封装）
- [ ] 确认所有依赖版本兼容

### 步骤1.3：创建application.yml配置
- [ ] 配置服务器端口（默认8080）
- [ ] 配置MySQL连接信息（url、username、password）
- [ ] 配置Redis连接信息（host、port、password）
- [ ] 配置MyBatis-Plus（mapper-locations、type-aliases-package）
- [ ] 配置日志级别和格式

### 步骤1.4：创建common/config包 - 配置类
- [ ] 创建`MyBatisPlusConfig.java`
  - 配置分页插件
  - 配置逻辑删除插件
- [ ] 创建`RedisConfig.java`
  - 配置RedisTemplate序列化（StringRedisSerializer + Jackson2JsonRedisSerializer）
  - 配置连接池参数
- [ ] 创建`ThreadPoolConfig.java`
  - 配置全局线程池（核心4，最大7，有界队列100）
  - 配置自定义拒绝策略

### 步骤1.5：创建common/result包 - 统一响应
- [ ] 创建`Result.java`
  - 字段: code, message, data
  - 静态方法: success(), success(data), error(message), error(code, message)

### 步骤1.6：创建common/exception包 - 异常处理
- [ ] 创建`BusinessException.java`
  - 继承RuntimeException
  - 字段: code, message
- [ ] 创建`GlobalExceptionHandler.java`
  - @ControllerAdvice注解
  - @ExceptionHandler处理BusinessException
  - @ExceptionHandler处理Exception（兜底）

### 步骤1.7：创建common/constants包 - 常量
- [ ] 创建`SystemConstants.java`
  - 定义系统常量（在线状态枚举、LED状态枚举等）

### 步骤1.8：创建数据库
- [ ] 创建数据库`edge_video_analysis`
- [ ] 执行建表SQL（见阶段1.9）
- [ ] 创建索引

### 步骤1.9：创建数据库表
- [ ] 创建`t_lamp`表（灯杆设备表）
- [ ] 创建`t_sensor_data`表（传感器数据表）
- [ ] 创建`t_alarm_rule`表（报警规则表）
- [ ] 创建`t_alarm_record`表（报警记录表）

- [ ] 创建`t_control_command`表（控制指令表）
- [ ] 创建所有索引（lamp_id + time复合索引等）

### 步骤1.10：验证基础框架
- [ ] 启动Spring Boot应用
